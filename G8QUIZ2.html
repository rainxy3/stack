Relations and Functions

1. It is any set of ordered pairs.

a. Rectangular Coordinate System

b. Function

c. Relation

d. Diagram


2. It is a relation in which every element in the domain is mapped to exactly one element in the range.

a. Rectangular Coordinate System

b. Function

c. Relation

d. Diagram


3. It consists of two real number lines that intersect at a right angle.

a. Rectangular Coordinate System

b. Function

c. Relation

d. Diagram


4. Which of the following relations is NOT a function?

a. {(1,-5), (3,1), (-5,4), (4,-2)}

b. {(2,7), (3,7), (4,7), (5,8)}

c. {(3,-2), (5,-6), (7,7), (8,8)}

d. {(1,-5), (-1,6), (1,5), (6,-3)}


5. For the function {(0,1), (1,-3), (2,-4), (-4,1)}. Write the domain of the function.

a. D = {1, -3, -4}

b. D = {0, 1, 2, -4}

c. D = {1, -3, -4, 1}

d. D = {0, 1, 2, -4, 1}


6. For the function {(0,1), (1,-3), (2,-4), (-4,1)}. Write the range of the function.

a. R = {1, -3, -4}

b. R = {1, -3, -4, 1}

c. R = {0, 1, 2, -4}

d. R = {0, 1, 2, -4, 1}


7. Which of the following is NOT a function?

a. {(0,1), (1,2), (2,3), (3,4)}

b. {(1,2), (2,2), (3,2), (4,2)}

c. {(1,3), (4,2), (2,0), (3,4)}

d. {(0,2), (1,3), (4,3), (1,2)}


8. A relation pairs each input with exactly one output is a ____.

a. Similarity

b. Congruent

c. Function

d. None of the above


9. All of the y values or outputs are called what?

a. Function

b. Relation

c. Domain

d. Range


10. All of the x values or inputs are called what?

a. Function

b. Relation

c. Domain

d. Range
