Polynomial Functions

1. It is a function that can be expressed in the form of a polynomial.

a. Quadratic Function

b. Linear Function

c. Polynomial Function

d. None of the above


2. Which of the following expressions is an example of polynomial?

a. 4x^1/2+3x+1

b. 3x-7

c. (9x +1) ÷ (x)

d. None of the above


3. Which of the following expressions is NOT an example of a polynomial?

a. 7x^3+x2-2

b. x^2+2x+1

c. 5x^-1+1

d. None of the above


4. It is a constant polynomial function whose value is zero.

a. Zero Polynomial Function

b. Linear Polynomial Function

c. Quadratic Polynomial Function

d. Constant Polynomial Function


5. A polynomial function with a degree of 0 is called what?

a. Zero Polynomial Function

b. Linear Polynomial Function

c. Quadratic Polynomial Function

d. Constant Polynomial Function


6.What type of function is f(x) = 5x^3 + 2x^2+ 3x + 7?

a. Linear Polynomial Function

b. Cubic Polynomial Function

c. Quintic Polynomial Function

d. Quartic Polynomial Function


7. A polynomial function with a degree of 1 is called what?

a. Zero Polynomial Function

b. Linear Polynomial Function

c. Quadratic Polynomial Function

d. Constant Polynomial Function


8. A polynomial function with a degree of 2 is called what?

a. Zero Polynomial Function

b. Linear Polynomial Function

c. Quadratic Polynomial Function

d. Constant Polynomial Function


9.The standard form of a quadratic functions is P(x) = an x^n + an-1 x^n-1+.……….…+ a0.

a. True

b. False

c. Maybe

d. None of the above


10. What type of function is f(x) = 6x^2 + 4 - 3x^4 + 5x - 9x^3?

a. Quadratic Polynomial Function

b. Linear Polynomial Function

c. Quintic Polynomial Function

d. Quartic Polynomial Function
