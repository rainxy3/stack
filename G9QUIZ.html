MATHEMATICS | GRADE 9

Quadratic Functions

1. It is where the function has a maximum value or a minimum value.

a. Quadratic Function

b. Vertex

c. Parabola

d. Standard Form


2. The vertex form of a quadratic function is:

a. f(x) = a(x - p)(x - q)

b. f(x) = a(x - h)^2 + k

c. f(x) = ax^2 + bx + c

d. None of the above


3. It is a polynomial function with one or more variables in which the highest exponent of the variable is two.

a. Binomial

b. Quadratic Function

c. Quartic Function

d. Standard Form


4. The intercept form of a quadratic function is:

a. f(x) = a(x - p)(x - q)

b. f(x) = a(x - h)^2 + k

c. f(x) = ax2 + bx + c

d. None of the above


5. The curve in the graph of a quadratic function is called what?

a. Vertex

b. Quadratic Function

c. Standard Form

d. Parabola


6. The standard form of quadratic function is:

a. f(x) = a(x - p)(x - q)

b. f(x) = a(x - h)^2 + k

c. f(x) = ax^2 + bx + c

d. None of the above


7. What is the y-intercept for the equation: y = -8x^2 + 3x – 7

a. (0, 3)

b. (0, -8)

c. (0, 7)

d. (0, -7)


8. Given the equation y = 3(x + 5)^2 - 4, what is the vertex of the parabola?

a. (-5, -4)

b. (5, -4)

c. (-15, -4)

d. (15, -4)


9. Find the missing value to create a perfect square trinomial: x^2 – 8x + ____.

a. 4

b. -4

c. 16

d. -16


10. What are the x-intercepts of the function f(x) = (x -2)(x + 3)?

a. x = -2, x = 3

b. x = 2, x = -3

c. x = 2, x = 3

d. There are no real roots
