MATHEMATICS | GRADE 7

Sets

1. An object or member in a set is called ____

a. an item

b. a unit

c. an element

d. a product


2. What type of set is denoted as either {} or φ?

a. Universal Set

b. Null Set

c. Parenthetical Set

d. Cardinality of the Set


3. It is a well-defined group of objects, called elements that share a common characteristic.

a. Element

b. Universal Set

c. Set

d. Union


4. If set A = {a, b, c, d, e} and set B = { b, c, d}, what kind of relationship does set A and B have?

a. Subset

b. Proper Subset

c. Roster Form

d. Set Builder Form


5. It is a method that is used to express sets with an interval or an equation.

a. Subset

b. Proper Subset

c. Roster Form

d. Set Builder Form


6. It represents the elements of a set in a row surrounded by curly brackets and separates elements by commas.

a. Subset

b. Proper Subset

c. Roster Form

d. Set Builder Form


7. If set A = {2, 4, 6, 8, 10} and set B = {5, 10, 15, 20}, what is the intersection of set A and set B (A ∩ B)?

a. A ∩ B = {2, 4, 6, 8, 10, 5, 15, 20}

b. A ∩ B = {2, 4, 6, 8}

c. A ∩ B = {10}

d. A ∩ B = {2, 4, 6, 8, 5, 15, 20}


8. Let set A = {2, 4, 6, 8, 10} and set B = {5, 10, 15, 20}, what is the cardinality of set A?.

a. n(A) = 5

b. n(A) = 4

c. n(A) = 9

d. n(A) = 7


9. Let U = {1, 2, 3, 4, 6, 7, 8, 9, 10} and A = {1, 2, 3}, what is the complement of set A or A’?

a. A’ = {4, 5, 6, 7, 8, 9, 10}

b. A’ = {1, 2, 3}

c. A’ = {4, 6, 7, 8, 9, 10}

d. A’ = {9, 3}


10. Let A = {a, b, c, d, e} and B = {d, e, f, g, h, i}, what is A union B or A ∪ B?

a. A ∪ B = {a, b, c, d, e, f, g, h, i}

b. A ∪ B = {a, b, c, d, e, d, e, f, g, h, i}

c. A ∪ B = {a, b, c}

d. A ∪ B = {f, g, h, i}
